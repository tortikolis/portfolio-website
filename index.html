Project Overview

BitBook

Bitbook is Facebook inspired social network app. Users are able to register and log in. They are also able to make tree different kind of posts: image, video and text. Commenting on posts is also supported. There are also sections about site users and their profiles which could be edited. React.js is for building user interface and Materialize is used for design.

Project Overview

TVShows

TVShows is a site that provides info about TV shows. It allows users to search for the series and displays detailed info about searching show (poster, plot, seasons, rating, actors etc.). It also provides a daily schedule of shows on TV. App communicates with the TV Maze API.
